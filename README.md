# todo-webapp
Todo web app using JSP, Servlet, MySQL, JDBC

App Features:
Register, 
Login, 
Add todo, 
Edit todo, 
Delete todo, 
List todo.

Configration:
Java 11, 
JSP, 
Servlet, 
MySQL 5.5, 
STS, 
apache-tomcat-8.5.87.
